## 📖 Apa Ini?
Ini adalah tool all-in-one buat ngoprek performa Windows (10/11) biar enteng, ngebut, dan anti lemot.  
Dibuat oleh Aethernal – tempatnya para pejuang Laptop Kentang.

Cocok buat:
- Laptop kentang yang pengen main Wuthwering Waves Rata Galaxy
- PC kantor yang lemot abis install bloatware
- Lo yang males ngutak-atik registry sendiri
- Bocil kerasukan RTX

---

## ✨ Fitur-Fitur Gelooo !!!!

| Menu | Fungsi | Efek Samping |
|------|--------|---------------|
| [1] Cache Cleaner | Bersihin sampah sistem + browser | 👉 Laptop lega, napas panjang |
| [2] SFC Check | Benerin file Windows yang rusak | 👉 Error ilang, stabilitas naik |
| [3] Core Optimize | Maksimalin CPU, GPU, RAM | 👉 FPS naik, multitasking mulus |
| [4] LOW-END Mode | Matiin efek-efek gak penting | 👉 Cocok buat spek rendah (<4GB RAM) |
| [5] OPTIONAL Optimize | Hapus bloatware, matiin telemetry, ganti DNS | 👉 Bebas iklan, koneksi stabil |
| [6] Reset All Tweaks | Kembalikan semua ke default Windows | 👉 Aman buat yang takut salah |

## 🛠️ Cara Pakai (Tutorial Paling Gampang)

### 📥 1. Download
Klik tombol hijau "Code" di kanan atas → "Download ZIP"  
Atau clone pake:
`bash
git clone [https://github.com/Aethernal07/Jendela-Optimizer-Tweak.git]


🖱️ 2. Jalankan Sebagai Administrator

⚠️ WAJIB! Klik kanan file GACOR_STORE_Optimizer.bat → Run as administrator
Kalo gak, fitur-fitur tertentu gak bakal jalan.

🎮 3. Pilih Menu

Nanti muncul tampilan kayak gini:
===============================================
        AETHERNAL - PC OPTIMIZER v3.0
              Windows 10 / 11
===============================================

[ DAILY USE ]
  [1] Cache Cleaner
  [2] SFC Check

[MAXIMIZE PERFORMANCE]
  [3] Core Optimize
  [4] LOW-END Mode
  [5] OPTIONAL Optimize
  [6] Reset All Tweaks

  [0] Exit

  
Contoh penggunaan:

· Mau bersihin cache? Tekan 1 → Enter
· Mau optimasi core? Tekan 3 → Enter
· Mau reset semua? Tekan 6 → Enter

✅ 4. Restart (Disarankan)

Setelah selesai, restart laptop biar perubahan maksimal.

⚠️ Peringatan (Biar lo gak nyesel)

· Script ini aman asal di PC Yang ga bluescreen.
· Jangan coba-coba di warnet atau laptop orang tanpa izin.
· Untuk menu OPTIONAL Optimize, baca dulu sub-menunya sebelum pencet.
· Kalo ada error, tinggal pilih [6] Reset All Tweaks dan restart.
